xxm Demo Projects - 05 Image

This is a demo project showing how to respond to a request with a dynamically generated image.

Web.xxmp
The project file, contains an internal list of the files of the project, and other data about the project.
Use xxmProject.exe to edit and manage project settings.
Default.xxm
A TBitmap object is created, and it's Canvas used to draw on it. A TJPEGImage object is used to encode the bitmap in JPEG format. A TxxmOutputStream object is used to output this data. Note that first the response's MIME type is set using Context.ContentType.
