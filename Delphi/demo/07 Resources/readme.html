xxm Demo Projects - 07 Resources

This is a demo project showing how to include files into the resources of the xxl module, and respond to requests for these files with data from these resources. This way all data for a website is contained in a single file, which enables easy (auto-)updates.

Web.xxmp
The project file, contains an internal list of the files of the project, and other data about the project.
This Web.xxmp has extra entries for actions to perform to prepare the resource files:
<PreCompileCommand>prebuild\prepres.exe</PreCompileCommand>
Lists files in the res\ folder and generates the resources list
<Switches>{$R '..\Web.res' '..\Web.rc'}</Switches>
Includes the Pascal code to include the resource file into the compiled library
<PreCompileCommand>brcc32 -32 Web.rc</PreCompileCommand>
Calls the resource compiler on the resource script
xxmp.pas
The project's main unit. When this file is missing form the project, xxm creates one for you.
This one has been modified to divert all requests to the res\ folder to a dedicated TxxmPage descendant that serves the requested file from resource data.
prebuild\
Contains the source-code for the PrepRes utility that lists the files in the res\ folder and generates the resource script.
res\
Contains the files to include into module resources.
Default.xxm
The website's default page. This simple example uses files from resources by using relative URL's.
