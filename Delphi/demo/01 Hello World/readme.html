xxm Demo Projects - 01 Hello World

This is a basic demo project showing the basic files of an xxm project.

Web.xxmp
The project file, contains an internal list of the files of the project, and other data about the project.
Use xxmProject.exe to edit and manage project settings.
xxmp.pas
The project's main unit. When this file is missing form the project, xxm creates one for you.
Default.xxm
The website's default page. It contains the code of a basic HTML page, a hyperlink to Hello1.xxm and the Delphi code to output the string 'Hello world'.
Hello1.xxm
A second page of the website. It contains code to include Chunk.xxmi a number of times. If no count parameter value is provided, it is included 10 times. You can define a different count value by appending a query string to the URL, e.g.: xxm://demo/Hello1.xxm?count=20
Chunk.xxmi
An include file. It displays the value that's passed to it, followed by "#: Hello World!"
