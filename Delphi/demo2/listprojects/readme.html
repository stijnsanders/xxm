xxm Demo Projects - listprojects

This is a demo project listing the projects from a xxm.xml file.

Web.xxmp
The project file. Use xxmProject.exe to edit and manage project settings.
xxmp.pas
The project's main unit. When this file is missing form the project, xxm creates one for you.
MSXML2_TLB.pas
The wrapper unit for the Microsoft XML library, used to read the xxm.xml file.
Default.xxm
The website's default page. It searches the directory listprojects.xxl is in, and the directories above it, for the xxm.xml file. Loads it, and lists the <Project> tags by name (skipping any alias definitions and "xxm").

By default, a request for the root page on an xxm handler will forward to "/xxm/" (unless a SingleProject is configured in the project registry). Register this project as ProjectName="xxm" to offer a list of available projects.

Important: Use this for development servers or environments where it's safe to publish the full list of available projects on the host.
