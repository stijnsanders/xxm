xxm Demo Projects - 09 Upload Progress

This is a demo project showing how show the progress of large file uploads.

Web.xxmp
The project file, contains an internal list of the files of the project, and other data about the project.
Use xxmProject.exe to edit and manage project settings.
xxmp.pas
The project's main unit. When this file is missing form the project, xxm creates one for you.
This one has been modified to include the SetSession(Context); call on LoadPage, linking each request to a new or existing session.
A session is needed to link incoming file uploads with requests to query the upload progress.
xxmSession.pas
This unit defines the TxxmSession class, and SetSession and AbandonSession procedures.
This demo only uses the session to store upload progress information.
iHead.xxmi
An include file for pages to use with the HTML header
iFoot.xxmi
An include file for pages to use with the HTML footer
demo9.css
The website's style sheet, linked to by iHead.xxmi, and thus by any page that includes iHead.xxmi.
files\
Directory uploaded files will be stored in. Attention: storing uploaded files within a directory of the website poses a security risk, especially when sensitive information is passed between pages using client side scripting.
Default.xxm
The website's default page. It displays a list of the files that have already been uploaded, and a link to the upload form.
fAdd.xxm
The upload form. An iframe element is used to display the upload progress when the form is posted
aAdd.xxm
Accepts the upload form post. An upload progress report agent is attached to the context.
Progress.xxm
Displays the upload progress.
